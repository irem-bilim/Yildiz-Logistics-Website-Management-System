#  Yıldız Nakliyat Web Sitesi

Yıldız Nakliyat için geliştirilmiş modern ve dinamik bir kurumsal web sitesidir.
Proje, şirketin hizmetlerini tanıtmak, araç filosunu göstermek ve müşterilerle iletişim kurmasını sağlamak amacıyla geliştirilmiştir.

Site hem kullanıcı arayüzü hem de yönetim paneli içermektedir. Yönetim paneli sayesinde site içerikleri kolayca güncellenebilir.

---

##  Özellikler

* Modern ve responsive (mobil uyumlu) tasarım
* Ana sayfa slider (intro)
* Hakkımızda bölümü
* Hizmetlerimiz bölümü
* Araç Filosu galerisi
* Referanslar slider
* Müşteri yorumları
* İletişim formu
* Google Maps entegrasyonu
* Yönetim paneli ile içerik yönetimi

---

##  Kullanılan Teknolojiler

Frontend:

* HTML5
* CSS3
* Bootstrap
* JavaScript
* jQuery
* Owl Carousel
* WOW.js

Backend:

* PHP
* MySQL
* PDO

Araçlar:

* Visual Studio Code
* WampServer
* phpMyAdmin
* Git & GitHub

---


##  Yönetim Paneli

Site içeriklerini yönetmek için admin paneli bulunmaktadır.

Panel üzerinden yönetilebilen alanlar:

* Site ayarları
* Intro slider
* Araç filosu
* Hizmetler
* Referanslar
* Hakkımızda bölümü
* Müşteri yorumları
